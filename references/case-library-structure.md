# 案例库目录建议

```text
references/
├── prompts.md
├── case-feeding-template.md
├── anti-pattern-template.md
├── cases/
│   ├── business-opportunity/
│   ├── commercialization/
│   ├── platform-adaptation/
│   ├── spec-definition/
│   └── custom-revamp/
└── anti-patterns/
```

## 说明

- `cases/business-opportunity/`：复杂业务型正例
- `cases/commercialization/`：商业化优化类正例
- `cases/platform-adaptation/`：平台适配 / SDK 类正例
- `cases/spec-definition/`：卡片 / 规格定义类正例
- `cases/custom-revamp/`：定制改造类正例
- `anti-patterns/`：反例库

## 命名建议

统一格式：

`YYYYMMDD-类型-案例名.md`

示例：
- `20260312-business-opportunity-车载一起听一期.md`
- `20260407-commercialization-车载商业化触点优化.md`
- `20260615-platform-adaptation-鸿蒙单框架媒体适配SDK.md`
- `20260511-spec-definition-鸿蒙双框架桌面卡片.md`
- `20260612-custom-revamp-无麦K歌定制版.md`
